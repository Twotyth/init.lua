# init.lua
My (probably bad) tries to setup Neovim for my liking

## Plugins
My main goal for this config was to create very lightweight and stable environment to create code efficiently and comfortably.
Here is list of installed plugins :
  * [Packer] - My plugin manager of choice
  * [Web dev icons] - Just better
  * [Telescope] - Fuzzy search
  * [Harpoon] - Quick navigation between chosen files
  * [Tint] - Visual window split / floating window focus
  * [Trouble] - Better diagnostics, quickfix, all the good stuff
  * [Tree sitter] - Required plugin to even work in neovim
  * [Lsp zero] - Configure and manage language servers easily
  * [Lualine] - Status line of my choice. Like the visual differentiation between input modes
  * [Autopairs] - Insert paired characters and auto-manage tabulation on new line. Handy
  * [Project] - Project manager on opening some source file
